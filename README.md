# read-films

## Usage
First, clone the repository locally:
```
$ git clone https://github.com/fskjs/read-films.git
```
To install all required dependencies, run:
```
$ pip install -r requirements.txt
```
## Data Preparation
We use the "./dicompro.py" file to extract all period of dicom files to one folder.

## Training
Use the following command to train 5-fold cross-validation models.

```
python train.py
```
